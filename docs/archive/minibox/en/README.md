# CoreBOX

The Discovery Core Box is the foundation of the UC2 Discovery Series, perfect for classrooms, workshops, and self-learners exploring the basic principles of optics.

## What You'll Learn

- Basic principles of geometric optics
- How lenses work and focal length concepts
- Building a telescope from scratch
- Creating a simple microscope
- Puzzle-based assembly of optical components

## Experiments in this Section

### 1. Core Introduction
Get started with the CoreBox and understand the modular UC2 system. Learn how puzzle-based assembly makes optics accessible and fun.

### 2. MiniBox Introduction
Explore the compact version of the CoreBox for portable optical experiments.

### 3. Understanding Lenses
Learn about focal lengths, lens properties, and how to measure them experimentally.

### 4. Building a Telescope
Construct a working telescope using UC2 components and understand how magnification works.

### 5. Building a Microscope
Create your own microscope to observe tiny structures and learn the principles of magnification.

### 6. Smartphone Microscope
Transform your smartphone into a powerful microscope using simple optics.

### 7. Troubleshooting
Common issues and solutions when building optical setups.

### 8. Showcase
Examples of projects and experiments built with the CoreBox.

## Learning Objectives

- Understand focal length and lens equations
- Learn ray tracing and optical paths
- Master puzzle-based modular assembly
- Experiment with real optical components
- Build confidence in optics fundamentals

Perfect for beginners in optics, students, teachers, and anyone curious about how lenses and optical instruments work!
