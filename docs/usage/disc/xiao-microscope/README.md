# openUC2 XIAO Microscope (Seeed Collaboration)

The openUC2 XIAO Microscope is a revolutionary collaboration with Seeed Studio that makes high-quality, automated microscopy accessible to everyone. This compact, powerful device is ideal for hackers, optics newcomers, and biologists, offering more functionality than standard USB microscopes.

## Key Features

### Powerful Camera
Equipped with ESP32-S3 and OV2560 sensor for high-resolution image streaming. Supports Arduino and CircuitPython programming.

### Micrometer-Precise Focus
Metal-based focusing mechanism ensures precise objective lens movement with support for motorized focus stacking and automation.

### Robust Power Design
Solid housing ensures reliability for everyday field research and classroom use.

### Swappable Objectives
Easily change magnifications with standard RMS-compatible microscope objectives to observe samples from different angles.

### Customized Firmware
Utilizes TinyML for on-the-fly image processing directly on the MCU. Integrates sensors and hardware for automated experiments.

## Technical Specifications

- **Camera:** ESP32-S3 with OV2560 sensor
- **Focus:** Motorized with precise metal mechanism
- **Power:** USB-C, supports battery operation
- **Storage:** SD card for autonomous image storage
- **Objectives:** RMS-compatible, swappable
- **Programming:** Arduino, CircuitPython, custom firmware
- **Size:** Compact enough to fit in a backpack

Perfect for educators, students, field researchers, and makers who need a portable, powerful, and programmable microscopy solution!
