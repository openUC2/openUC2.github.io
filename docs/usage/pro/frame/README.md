# FRAME

Welcome to the usage documentation for the openUC2 FRAME robotic microscope! Our documentation is organized into:

- [Hands-on tutorials](./tutorials/README.md) to help you learn how to use the FRAME in an effective way.

  :::tip

  If this is your first time setting up a FRAME, please continue to our [day-1 tutorial](./tutorials/day-1/README.md), where you will learn how to deploy and use your FRAME.

  :::

- [How-to guides](./guides/README.md) to help you accomplish practical goals with the FRAME, after you've finished our tutorials.

- [Reference guides](./reference/README.md) which you can consult for technical descriptions about our products & projects and how to use them

- [In-depth explanations](./explanations/README.md) to help you understand the FRAME's design, from the perspective of a user or operator of a FRAME machine.

  :::tip

  If you don't have a FRAME yet and you're not sure whether a FRAME is the right solution for you, please continue to our [explanations](./explanations/README.md) of the FRAME's design and what problems it solves.

  :::
